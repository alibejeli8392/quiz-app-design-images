# quiz-app-design-images
This repository contains the **initial design screenshots** of the QuickQuiz web application.  
QuickQuiz is a lightweight, fun, and non-competitive web app designed for users to take short quizzes for entertainment and self-reflection.  

---

## Screens Included

1. intro.png** – Intro & Welcome Screen  
   - Animated welcome text  
   - Buttons: "Start" and "Continue as Guest"  

2. login.png** – Login / Sign Up  
   - User login form  
   - Sign up form with username, email, and password  
   - Guest login option  

3. home.png** – Home Dashboard  
   - Greeting with user name  
   - Navigation sections at the bottom: Home, Topics, Profile, Explore  
   - Main sections: Trending quizzes, Recent Activity  

4. topics.png** – Topics Sections  
   - Multiple quiz topic sections  
   - Each section contains 4 topic cards  
   - Includes topic name, description, number of questions, and estimated duration  

5. profile.png** – Profile (Placeholder)  
   - User stats  
   - Achievements (locked/unlocked)  
   - Recent activity  

6. tips-fun-facts.png** – Tips / Fun Facts Section  
   - Non-competitive, engaging content  
   - Fun facts about quizzes and self-reflection
